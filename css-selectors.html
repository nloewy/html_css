CSS Selectors

About CSS Selectors

CSS Selectors are used to determine which html elements or set of elements to apply the CSS declarations too. The browser uses the selector API to traverse the document element model and pick out the elements matching the selector

It's good to be able to write a good selector, as it's also used in javascript as well

Types of Selectors

  1. Element Selector
    2. Says that every element of that type should be styled that way. For example `p{color:blue}`means every paragraph should be colored blue. Div elements however would not be affected by the rule
  2. Class Selector
    3. Formatted by `.name`
    ```css .blue { color: blue; } ```
    The above makes a class called blue. Then for each element you want the `blue` class to apply to, we give it that attribute, as shown below
    < pclass = "blue" > ... </p >
  3. ID Selector
    4. Specify an ID selector by doing a # sign and then the value of an id of an element in the html document
    Example writing an ID selector:
    #name { color:blue;}
    Example using the id selector
    < div id="name" > ... </div>
We can group selectors by making them many types of selectors:
Ex: `div, .blue { color:blue; }
