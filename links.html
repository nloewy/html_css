(Internal) Links

  • We link with the <a> tag
  • The a tag has an href attribute, which can be an absolute or relative path
  • The a tag has a title attribute, which is useful for screen readers for visually impaired users
  • Content between opening and closing a tags is what is actually displayed

Note: The <a> tag is both an inline and block element, so it can contain block elements within it (e.g., if you want a clickable region).
